## Organization Name Rules

- **Full name:** Marketing + Media Alliance
- **Always use the plus (+) symbol**, never ampersand (&) or "and"
- **First mention in any text:** "Marketing + Media Alliance (MMA)"
- **Subsequent mentions:** "MMA" or "Marketing + Media Alliance"
- **Legal name:** MMA Global, Inc. -- do NOT use in marketing copy
- **Not acceptable:** "MMA Global", "Marketing + Media Alliance (MMA Global)", "MMA Alliance", "The Alliance" (exception: "Join the Alliance" CTA and tagline)
- **Tagline:** "Come for the Alliance, Stay for the Results"
- **Regional format:** "Marketing + Media Alliance APAC (MMA APAC)" on first mention, then "MMA APAC"
- **Events must include brand name in title.** "MMA" can be used with event names (e.g., "MMA's Decoding AI for Marketers") as long as full name appears when the org is first mentioned.

## Document Formatting (Word / .docx)

When generating Word documents for MMA, apply these standards:

### Fonts
- **Headings:** Aptos Bold (substitute for Söhne Halbfett; Söhne is not available in Word)
- **Body text:** Aptos 12pt (substitute for Söhne Licht)
- **Pull quotes or serif distinction:** Georgia (substitute for Tiempos Text)
- **Never** use Calibri, Times New Roman, Arial, or other off-brand fonts

### Heading Hierarchy
- **Title / Document name:** Aptos Bold, 24-28pt, black
- **Heading 1:** Aptos Bold, 18-20pt, black
- **Heading 2:** Aptos Bold, 14-16pt, black or MMA Gold (#FFA400)
- **Heading 3:** Aptos Bold, 12pt, black
- **Body text:** Aptos, 12pt, black
- **Footnotes / captions:** Aptos, 10pt, grey

### Layout
- **Margins:** 1 inch all sides (standard)
- **Line spacing:** 1.15 for body text, 1.0 for headings
- **Paragraph spacing:** 6pt after paragraphs
- **Page numbers:** Bottom right, Aptos 10pt

### Accent Elements
- Use MMA Gold (#FFA400) for accent lines, horizontal rules, or heading highlights
- Use MMA Black (#000000) for primary text
- Tables: header row in MMA Gold background with white text, alternating light grey rows
- Avoid gradients on any brand element

### Cover Page (when applicable)
- Document title in Aptos Bold, 28-36pt
- Subtitle or description in Aptos, 14-16pt
- "Marketing + Media Alliance" or MMA logo
- Date and author
- MMA Gold accent bar or Diagonal element

### General Rules
- First mention of MMA: "Marketing + Media Alliance (MMA)"
- Follow all naming rules from the Organization Name Rules section above
- Include page numbers on multi-page documents
- Use bullet points for lists of 3+ items
- Bold key terms, lab names, and program acronyms on first mention

## Presentation Formatting (PowerPoint / .pptx)

When generating PowerPoint presentations for MMA, apply these standards.

### Template File (REQUIRED)

**Always start from the official MMA template** when creating new presentations:
- **SharePoint:** `MMA Internal - Documents/General/Marketing + Media Alliance MMA Brand Kit 2025/MMA PPT Template Dec 2025.potx`
- **Link:** https://mmaglobalcom.sharepoint.com/:p:/s/MMAGlobal/IQCGHSYJNOZCRq83EI8nvl5oASdduEMkRsSWTQ_BIF46qpo
- If a .pptx version is saved locally, use that. The template contains the correct slide masters, MMA logo placement, fonts, and brand elements.
- **In python-pptx:** Open the template with `Presentation('template.pptx')` so all slide layouts inherit the masters.
- **In Claude for PowerPoint add-in:** The template should already be loaded as the open deck.
- If no template is available, follow the rules below exactly.

### Slide Master Layouts

The MMA Core Gold master contains these layouts. Use them by name:

| Layout | When to use |
|--------|------------|
| **1_Title Slide** | First slide of every deck. White background, MMA logo top-left, gold diagonal right side. Title + subtitle + presenter info. |
| **1_Title and Content Black Bullets** | **Default for most content slides with custom shapes.** Has the title placeholder with gold accent bar, but NO body text placeholder. Use this when adding shapes, cards, charts, or tables instead of bullet text. This avoids leaving an empty body placeholder on the slide. |
| **Title and Content Gold Bullets** | Content slides where bullet-point text is the primary content. Title area at top, gold bullet content below. Use when the slide IS a list of bullets and no custom shapes are needed. |
| **Section Header** | Section divider slides. Gold background, black text. Title: 60pt Söhne Halbfett. Subtitle: 36pt Söhne Leicht. Either can be slightly smaller to fit. |
| **Title and Content 2/3** | When you need a larger content area |
| **Title and Content 1/3** | When content is minimal |
| **Two Content** | Side-by-side comparison slides (two body placeholders) |
| **Title and Content Black Bullets** | Full layout with body text placeholder (black bullets) |
| **Content with Caption** | Content with a sidebar caption |
| **Blank** | **Only for full-bleed images.** Never use as a general content slide. All content slides need the title placeholder and gold accent bar from the master. |

**Layout selection rule:** If you are adding custom shapes (cards, charts, tables, callout boxes) instead of plain bullet text, use **1_Title and Content Black Bullets** (no body placeholder to delete). If the slide is primarily a bulleted list, use **Title and Content Gold Bullets**. Never use Blank for regular content.

**Think Tank masters:** ALTT (Peridot), MATT (Emerald), MOSTT (Topaz), DATT (Sapphire) each have their own set of masters in the same template. Use the think tank's masters and colors when presenting that think tank's content.

### Title Slide Format (1_Title Slide)

The title slide uses the template's white background with MMA logo (top-left) and gold diagonal (right side).
- **Title:** Söhne Halbfett, large (40-48pt), black. Placed in the center-left area with the gold diagonal accent bar to its left.
- **Subtitle (if any):** Söhne Leicht, lighter weight, placed below the title in the same text box.
- **Event/date line:** Söhne Leicht, ~20pt, dark gray, placed below.
- **Presenter name:** Söhne Leicht, ~16pt, gray, below the date.

### Slide Title and Subtitle Format

- **Slide titles:** 36pt Söhne Halbfett, black, bold. Can be slightly smaller (32pt) if needed to fit, but never below 28pt.
- **Subtitles** are placed in the SAME text box as the title, as a second paragraph. Söhne Leicht, gray (#666666), slightly smaller than the title (18-20pt). Not a separate text box.
- Example: Title paragraph "Decoding AI for Marketers: 5-day training recap" in 36pt bold black, then a second paragraph "7th edition. Led by Rex Briggs (Mar 30 - Apr 3). 9,334 total registrations." in 18pt regular gray.

### Fonts

- **Headlines / titles / bold text:** Söhne Halbfett (preferred) or Aptos Bold (substitute)
- **Body text / bullets / subtitles:** Söhne Leicht (preferred) or Aptos (substitute)
- **Pull quotes / long-form / serif distinction:** Tiempos Text Regular (preferred) or Georgia (substitute)
- If Söhne or Tiempos are unavailable in the template, use the substitutes.

### Minimum Font Sizes (never go below)

- Slide titles: 28pt minimum (prefer 36pt default)
- Subtitles: 18pt minimum (prefer 18-20pt)
- Body text and bullets: 16pt minimum (prefer 18-20pt)
- Table cell text: 14pt minimum (prefer 16pt)
- Footnotes, tips, source citations: 12pt minimum
- **Never use anything below 12pt on any slide**

### Leading and Tracking

- Headlines: +10% leading, 0 tracking
- Body text: +20% leading, 0 tracking

### Colors

- Default slide background: white
- Section dividers: MMA Gold (#FFA400) background with black text
- Gold accent bars: #FFA400
- Text: black on white backgrounds, white on dark/gold backgrounds
- Never use gradients on logo or brand elements
- Ensure sufficient contrast: no light text on light backgrounds, no gold text on white

| **MATT** (Marketing Attribution) | MMA Emerald | `#00AB84` | Lt Emerald | `#66CDB7` |

| **DATT** (Data & CX) | MMA Sapphire | `#0047BB` | Lt Sapphire | `#4079D6` |

For non-Think-Tank presentations (most MMA content), use the primary gold palette. Secondary colors and their 75% tints can be used sparingly for:
- Data visualization series differentiation

- Flow diagram nodes

### Layout Rules

- One idea per slide
- Headlines should be takeaways, not topic labels ("CAP delivers +160% avg lift" not "CAP Results")
- **Slide titles never end in a period.** Questions marks are fine.
- Limit bullet points to 5-6 per slide maximum
- Vary slide layouts: don't use the same layout for more than 3 consecutive slides
- Two-column layouts: balance content between columns. If one column has significantly more content, use a different layout.
- Charts and diagrams: native PowerPoint objects (editable, not images)
- Icons: simple line-style, consistent weight, in MMA Gold, black, or Think Tank accent color
- Prefer generous white space over dense, crowded slides
- **Use shapes for visual richness.** The slide master provides the layout structure (title, footer, logo), but add custom shapes, cards, and visual elements on content slides. Do NOT render all content as plain bullet text. Combine the template's placeholders with additional shapes for engaging layouts.

### Slide Numbers

The template includes a slide number placeholder (lower-left corner, PH index 12).
- **Always populate slide numbers** on every slide.
- In python-pptx, set `slide.placeholders[12].text` to the slide number, or insert a field element.
- Position: approximately (0.30", 6.95"), matching the template's footer area.

### Section Dividers

- Use the "Section Header" master layout
- Full MMA Gold (#FFA400) background
- Black text
- MMA logo in bottom-right (white version, provided by the slide master)
- Use The Diagonal (gold slash) as brand accent on title and section slides

### Tables

Use PowerPoint's built-in table objects (not shapes arranged to look like tables).

- **Header row:** MMA Gold (#FFA400) background by default; also try dark gray (#333333) for a subtler look. Text: black on gold headers, white on dark headers. Avoid pure black (#000000) headers as they can be too heavy.
- **Data rows:** Alternating white (#FFFFFF) and visible light gray (#EBEBEB) backgrounds. The contrast must be clearly visible.
- **First column styling:** Use bold text (Söhne Halbfett) in the first column when it serves as a row label. Keep the cell background the same as the rest of the row (do NOT give the first column a different background color, as this creates an awkward grid effect).
- **Gold highlights:** When highlighting specific cell text in gold (#FFA400), use Söhne Halbfett for emphasis.
- **Divider lines:** Table borders are usually sufficient. Don't add extra divider lines inside tables unless there's a clear visual reason.
- **Text color:** Black or dark gray for most cells. Gold (#FFA400) is acceptable for important callouts.
- **Borders:** Thin border (#D0D0D0 or similar light gray) around the table and between cells
- **Cell padding:** Adequate internal margins so text doesn't touch cell edges
- **Font size:** 14pt minimum for cell text, 16pt preferred
- **Header text:** Bold, 15-16pt

### Visual Design: Shapes, Cards, and Accent Elements

**IMPORTANT: Use shapes to create visually engaging slides.** Don't just use bullet text from the placeholder. Combine the template's title placeholder with custom shapes in the content area.

#### General Shape Rules

- **All text inside shapes:** Use the shape's text frame with margins. Never overlay text boxes on shapes.
- **Center alignment:** Groups of shapes on a slide should be centered horizontally. A single shape should be centered. Multiple shapes in a row should be collectively centered.
- **Corner radius:** Use SUBTLE rounded corners on large cards (small radius, not dramatic curves). For small cards (under 3" wide), slightly more rounding is acceptable. The default ROUNDED_RECTANGLE radius is often too large; reduce it when possible.
- **75% tints preferred for card backgrounds.** Use Lt Gold, Lt Emerald, Lt Sapphire, Lt Topaz, Lt Peridot for card fills. Reserve fully saturated colors for text, accent bars, borders, and small highlight elements. Exception: black cards are fine with full saturation.
- **Top-align text** inside cards so content starts at a consistent position regardless of how many lines of body text there are.

#### Card Patterns

**Basic card:**
- Rounded rectangle (subtle corners), fill #F5F5F5 (light gray) or a 75% tint color
- Internal margins: 0.2-0.25" left/right, 0.15" top/bottom
- **Title text color:** Use black or white text for card titles. On light backgrounds (gray, Lt Gold, Lt Emerald, Lt Topaz, Lt Peridot), use black. On dark backgrounds (black, Lt Sapphire), use white. Do NOT use the accent color (e.g., gold on Lt Gold) for title text as it looks bad. Exception: gold titles on a standalone gray or black card are fine.
- **NEVER use same-hue text on same-hue background.** No blue text on blue cards, no gold text on gold cards, no green text on green cards. This is a universal rule across all slides, not just cards. Text must always contrast with its background.
- Body text in Söhne Leicht, dark gray or white depending on background.
- Cards should include a visual anchor: a numbered circle, an icon, or bold title. Plain gray cards with all-black text are too basic.
- **Shape sizing:** Card height should adapt to the amount of text content. Don't leave large empty space in cards.

**Card with accent bar (top or left):**
- The accent bar can go on the LEFT or TOP edge of the card.
- **Bar colors:** Use 75% TINT colors for bars (Lt Gold, Lt Emerald, etc.), not full-saturation colors. Title text uses full-saturation color for contrast.
- **The accent bar:** A plain RECTANGLE (0.07" wide for left bars, 0.07" tall for top bars).

**Top bar cards (preferred, best corner control):**
- Use `ROUND_2_SAME_RECTANGLE` (MSO_SHAPE value 152) with `shape.rotation = 180.0`. This gives sharp 90-degree TOP corners and subtly rounded BOTTOM corners.
- Set corner radius via adj1=5000, adj2=0 on the prstGeom avLst.
- Since rotation flips text inside shapes, place ALL text in a SEPARATE TEXT BOX positioned on top of the shape. Never put text inside a rotated shape.
- Place the bar RECTANGLE flush on the top edge of the card.

**Left bar cards:**
- Use a standard `ROUNDED_RECTANGLE` with tight corner radius (adj=5000). Place the left bar RECTANGLE on top (higher z-order), overlapping the card's left edge to mask the rounded left corners.
- Text can go inside the shape (not rotated) OR in a separate text box.
- The left corners won't be perfectly sharp (the bar covers most of the rounding). This is acceptable. For perfect sharp left corners, use the Claude PowerPoint add-in to edit shape points after generation.

**In Claude for PowerPoint add-in:** Edit the shape points directly to flatten the bar-side corners to right angles. This is more precise than programmatic approaches.

**Important python-pptx notes:**
- Do NOT use `custGeom`. Custom geometry paths render as invisible shapes.
- Do NOT rotate ROUND_2_SAME_RECTANGLE by 90 or 270 degrees. The width/height swap breaks bar alignment and card dimensions. Only 180-degree rotation works reliably.
- Prefer top bar cards over left bar cards when possible, since top bars have perfect corner geometry.

**Numbered card (for agendas, grids):**
- Gold circle (oval, MMA Gold fill, 0.45-0.55" diameter) with white number in Söhne Halbfett
- Place the number circle in the UPPER-LEFT corner of the card (overlapping the card edge slightly)
- Card body has the title + time/description below
- Use for agenda slides, process steps, ordered lists

**Grid/row layout with separators:**
- Vertical stack of items with thin horizontal lines (#E0E0E0, 1pt) between them
- Gold numbered circle on the left, title in bold, description in regular weight
- Good for agenda slides and step-by-step content

#### Callout Boxes

**Preferred callout style (light cream):**
- Rounded rectangle with very light gold/cream fill (~#FFF5E0 or very light tint of gold)
- Gold (#FFA400) border, 2pt
- Optional: gold-colored icon on the left side of the text (e.g., download arrow, lightbulb, info circle)
- Label (e.g., "Tip:", "Download:", "Schedule:") in Söhne Halbfett MMA Gold (#FFA400), INLINE with body text on the same paragraph. Left-aligned, not centered.
- Body text in Söhne Leicht, dark gray or black, same paragraph as label
- **Links in callout boxes:** Use Söhne Halbfett and apply a hyperlink.
- **Do NOT use gold text on gold/orange backgrounds** (unreadable). Labels should be black or dark gray on light backgrounds.

**Dark callout (for emphasis):**
- Rounded rectangle with black fill
- Label in Söhne Halbfett gold, inline with body text in white
- Example: "**Questions?** Reach out to Alec Foster or post in #ai-upskilling anytime."

**Key rules for callout text:**
- Labels and body text go on the SAME paragraph using mixed formatting (bold label run + regular body run)
- Never use gold/orange text on gold/orange backgrounds

#### Two-Column Comparison

Multiple approaches (vary across decks):
- **Accent bar on top (preferred):** Two cards side-by-side, each with a colored bar on top (different colors). Cards have sharp top corners and subtly rounded bottom corners. Add spacing between title and body text. Optional: thin horizontal divider line (as a shape overlaid on the card, not text) inside the card to visually separate sections.
- **Tint-filled cards:** Two cards side-by-side using different 75% tint backgrounds (e.g., Lt Sapphire vs Lt Gold). All corners subtly rounded. No accent bars needed; the color differentiates.
- **Internal divider lines:** When used, these should be shape objects (thin rectangles, ~1pt tall) placed on top of the card, horizontally centered with padding (~0.3-0.4") from the card edges. Not text underscores. Color: use a color that contrasts with the card background. On light cards, use #E0E0E0 or similar gray. On dark/colored cards (blue, black), use white or a lighter shade of the card color.
- **Column titles** in their respective accent colors (full saturation for text, e.g., Sapphire for title text on a light background).

#### Big Number / Stat Highlights

- Large number in Söhne Halbfett 60-80pt
- **Top-align text** so the number is always in the same position regardless of how many lines of supporting text follow
- Supporting text below in 18-22pt
- For 3 or fewer stat cards: use Gold, light gray (#F5F5F5), and/or black backgrounds. Make sure adjacent cards are visually distinct (don't use gold + lt gold side by side).
- For 4+ stat cards: use the 75% tint palette or accent colors to differentiate

#### Flow Diagrams

- Use rounded rectangle nodes with 75% tint fills for process steps
- Connect nodes with PowerPoint RIGHT_ARROW shapes (standard, not notched). Do not hand-build arrows from line + triangle combos.
- Arrows should have clear spacing from nodes (at least 0.15" gap on both sides)
- Label each node with step text inside the shape's text frame
- Optional: labels below each node for status/timing

#### Icons

- When adding icons to slides, use simple line-style icons in a consistent weight
- Preferred color: MMA Gold, or the relevant accent/tint color matching the card
- Icons work well in: callout boxes (left of text), card headers, numbered items (replacing circles), section dividers
- Keep icons small and decorative; they support the text, not replace it

### MMA Logo Awareness

The slide masters place the MMA logo in the lower-right corner of most layouts.
- **Do not place shapes or text boxes that overlap the logo area** (approximately the bottom-right 1.5" x 0.8" of the slide).
- If a slide has dense content or a large table that extends to the bottom, add a white rectangle behind all content to cover the logo, OR ensure your content stops above the logo area.

### Slide Deletion Safety

ALWAYS read/inspect slide content before deleting any slide. Never assume slide order from memory. Verify with list_slide_shapes or read_slide_text first.

### Images in Slides

- When possible, include relevant images to make slides more engaging.
- **In Claude Code:** Use Gemini MCP (`generate_image`) for AI-generated visuals, or web search + download for stock/reference images. python-pptx can insert any local image file.
